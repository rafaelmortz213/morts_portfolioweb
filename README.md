# Morts Portfolio System

A modern, professional, and highly scalable portfolio website for full-stack developers. Built with vanilla JavaScript for maximum performance and customizability.

##  Features

- **Unique Design**: Terminal-inspired futuristic UI with hexagonal profile display
- **Dynamic Projects**: JSON-driven project system with detailed modal views
- **Image Gallery**: Multi-image support with carousel navigation
- **Animated Skills**: Progressive skill bars with scroll-triggered animations
- **Fully Responsive**: Optimized for all devices and screen sizes
- **Performance**: Lazy loading, smooth animations, optimized assets
- **SEO Ready**: Semantic HTML structure
- **Modular Architecture**: Easy to maintain and extend

##  Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Inter, JetBrains Mono)
